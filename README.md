# FE 스타터 팩
FE 프로젝트 Easy-to-start


## Directories

```
    ./
    ├── src/                                # source directory
        ├── components/                     # components directory
            ├── *.html|css|ts|spec.ts       # component files
        ├── index.html                      # index file
        ├── main.ts                         # entry point
        ├── style.css                       # global styles
        ├── ...
    ├── karma.conf.js
    ├── package.json
    ├── webpack.config.js
    ├── tsconfig.json
    └── README.md
```


## Prerequisites
- Node.js (`^12.18.3`)


## Installation
```
npm install
```


## Usage


### Build app
```
npm run build
```

### Run app (deployed on [localhost:8080](localhost:8080))
```
# (with hot reloading)
npm start
# or (without hot reloading)
npm run server
```

### Run test (Karma)
```
npm test
```
